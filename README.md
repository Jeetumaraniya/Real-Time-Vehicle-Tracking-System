### Quick Start

1.  **Prerequisites**: Ensure Node.js and MongoDB are installed and running.

2.  **Backend Setup**:
    ```bash
    cd backend
    npm install
    # Create a .env file based on the documentation or use the default
    npm run dev
    ```

3.  **Frontend Setup**:
    ```bash
    cd frontend
    npm install
    npm start
    ```

4.  **Access**:
    - Application: `http://localhost:3000`
    - API: `http://localhost:5000`
